### NFT源码项目使用指南🧭

- 复制源码文件 nft_source_code.sol
- 粘贴到 http://remix.ethereum.org/
- 编译项目
- 部署项目（ETH、Polygon）
- 调用 activateSale() 开始mint
- 选择 mint 填写 wei的数量
